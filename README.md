## Структура проєкту

 `train_model.py` — скрипт для завантаження датасету MNIST, побудови, навчання та тестування згорткової нейронної мережі. Зберігає графіки навчання та готову модель.
 `app.py` — код інтерактивного вебзастосунку, створеного за допомогою Streamlit.
 `requirements.txt` — перелік необхідних Python-бібліотек для запуску проєкту.
 `mnist_cnn.h5` — файл збереженої навченої нейромережі (генерується післязапуску `train_model.py`).
 `plots/` — папка з графіками процесу навчання (Accuracy, Loss, Confusion Matrix).

##  Інструкція із запуску 

### 1. Вимоги до середовища
Переконайтеся, що на вашому комп'ютері встановлено **Python версії 3.10** або вище.

### 2. Встановлення залежностей
Відкрийте термінал у папці з проєктом та виконайте команду для встановлення всіх необхідних бібліотек:

pip install -r requirements.txt

### 3. Навчання моделі 
Якщо ви хочете самостійно перенавчити модель та згенерувати нові графіки, запустіть скрипт навчання. Процес може зайняти кілька хвилин:


python train_model.py

Після успішного завершення у папці з'явиться файл mnist_cnn.h5.

### 4. Запуск вебзастосунку
Для запуску графічного інтерфейсу виконайте команду:


streamlit run app.py

Ваш браузер автоматично відкриє сторінку за адресою http://localhost:8501.

 ## Використання застосунку

Намалюйте будь-яку цифру від 0 до 9 у чорному квадраті  за допомогою миші або тачпаду.

Натисніть кнопку "Розпізнати цифру".

Алгоритм комп'ютерного зору автоматично відцентрує вашу цифру, передасть її нейромережі та виведе результат на екран разом із графіком впевненості моделі. 

Щоб очистити поле, натисніть іконку кошика під малюнком. Щоб відмінити або відновити останню дію є стрілочки назад/вперід під малюнком.
